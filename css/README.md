Папка для скомпилированных стилей. Файлы препроцессоров держите в другой папке. Например, для *.less используйте папку less, для *.scss используйте папку sass и так далее.

Скомпилированные CSS-файлы не отправляйте в репозиторий. В репозитории должны быть только файлы препроцессора.

--

Структура файлов:

css/
| - style.css
less/
| - style.less
| - [другие *.less файлы]
sass/
| - style.scss
| - [другие *.scss файлы]
postcss/
| - style.css
| - [другие *.css файлы]
